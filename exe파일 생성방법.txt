###################### exe 파일 생성방법 ###########################
1.파이썬 3 설치 (설치되어 있다면 건너뛰기)
2.cmd 창 열어서 pyinstaller 설치 ->  pip install pyinstaller
3.cmd 창에서 파이썬 파일이 있는 디렉토리로 이동 ->  cd [파이썬 파일 경로] 
4.cmd 창에서 exe 생성 명령어 입력 -> python -m PyInstaller --onefile extract_table_name.py
